Highlight Features

Scdo is the world's largest scalable, consistent, decentralized blockchain.

High Throughput

Scdo transaction speed can be as high as 2000Tx/S, Sharding technology enables TPS can be increased as needed.

Mining

Open source platform, interchangeable Proof-of-Work consensus algorithms and all tokens are for network-supporting miner reward

Scalability

Scdo has its innovative sharding technology supporting inter-shard & intra-shard transactions in a high speed. Scdo network can be scaled up easily by increasing shard number as needed.

Heterogenous Forest

Scdo has not only a high TPS mainnet, but also higher TPS layer 2 Subchains; Heterogenous structure keep Scdo secure and higher Speed which can manifest all kinds of industry applications with security guaranteed.

Smart Contracts

Scdo is compatible with Ethereum EVM and Smart Contracts and Supports all kinds of industry applications

Anti-ASIC Conensus Algorithm

Innovative Anti-ASIC consensus algorithm emphasizing scientific calculation which well decentralizes networks and prevents computing power attack.
